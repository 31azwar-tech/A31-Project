# A31-PM

Personal project management PWA — tasks, subfolders, recurrence, reminders, activity log,
offline sync. Terhubung ke Firebase Firestore (project `a31-pm`).

**Live:** https://USERNAME.github.io/a31-pm/

## Fitur
- Kanban, Calendar, Timeline, Eisenhower Matrix, Dashboard, Activity Log
- Light / Dark mode
- Folder 2 level (grup → subfolder)
- Quick-add dengan parser bahasa Indonesia (`!high`, `#tag`, tanggal)
- Recurrence, reminder, snooze + notifikasi
- PWA: install ke home screen, bekerja offline

## Deploy
Situs statis — tidak ada build step. Lihat `TUTORIAL.md` untuk langkah lengkap.
